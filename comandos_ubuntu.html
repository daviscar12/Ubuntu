Comandos Básicos do Ubuntu

  • Início
  • Publicação de Páginas Estáticas
  • Ubuntu como Servidor
  • ls: Lista arquivos e diretórios no diretório atual.
  • cd: Navega entre diretórios.
  • mkdir: Cria um novo diretório.
  • rm: Remove arquivos ou diretórios.
  • sudo: Executa comandos com privilégios administrativos.
  • cp: Copia um arquivo para outro local
  • cp -r: Copia uma pasta com todo o conteudo para outro local
  • touch: cria um Arquivo
  • sudo apt update: Verifica se há atualizacoes a serem abaixadas.
  • sudo apt upgrade: Instala novas atualizações.
